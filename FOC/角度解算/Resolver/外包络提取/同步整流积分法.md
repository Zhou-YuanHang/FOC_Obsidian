本质上是利用载波周期内的积分来提取包络，而不是仅仅依赖单个采样点。它的思想是：对旋变输出信号在一个载波半周期或整个周期内进行积分，从而获得包络值。

**同步整流积分法**的步骤如下：

1. **同步整流**：在软件中生成一个与载波同频同相的方波信号（即符号函数 ![](https://cdn.nlark.com/yuque/__latex/3c11c575eb39d26a32e26f0c59edaa9c.svg)），将ADC采样得到的信号乘以这个方波。相当于把载波的正半周翻转到正极性，负半周也翻转到正极性。
2. **积分**：在载波的每个半周期（或整周期）内，对整流后的信号进行积分（累加）。
3. **结果**：积分得到的值正比于包络 ![](https://cdn.nlark.com/yuque/__latex/4fdb859c6718100ac03076b0aac60aa5.svg)。

在实际数字实现中，ADC以较高频率采样（比如每个载波周期采样数十点），然后在每个载波半周期内累加采样值，并根据当前载波极性对采样值取正或取负，最后累加结果就是包络的积分值。

### 2. 实现方式

#### 硬件前提

- ADC采样率需要足够高，通常建议每个载波周期采样 **8~16 个点** 以上，以保证积分精度。
- 采样触发可以自由运行（比如用定时器均匀触发），不需要精确对准载波峰值，这降低了时序对齐的难度。

#### 软件处理流程

1. **确定载波极性**：通过软件生成的载波相位或通过比较器检测，判断当前采样时刻的载波是正半周还是负半周。
2. **累加**：

- 在正半周，ADC采样值直接累加到积分变量。
- 在负半周，ADC采样值取负后再累加。

3. **复位**：每个载波半周期结束时，积分结果即为该半周期的包络积分值，送入后续角度计算，然后积分变量清零，开始下一个半周期的累加。
4. **可选平滑**：可以对连续多个半周期的积分结果进行平均或滤波，进一步抑制噪声。

#### 与峰值采样法的对比

|  |  |  |
| --- | --- | --- |
| 特性 | 峰值采样法 | 同步整流积分法 |
| 采样点数 | 每个载波周期1~2点 | 每个载波周期8~16点以上 |
| 时序要求 | 必须精确对准峰值，对硬件延迟敏感 | 对采样点相位不敏感，只需知道极性 |
| 抗噪性能 | 较差，单个采样点易受尖峰噪声影响 | 较好，积分平均作用可抑制噪声 |
| 计算量 | 极小 | 稍大（需要多次累加和极性判断） |
| 实现复杂度 | 需要精确的PWM同步触发 | 需要较高采样率，但同步逻辑简单 |

### 3. 为什么“同步整流积分法”有优势

在实际工程中，旋变信号可能包含：

- **高频噪声**（来自PWM开关、驱动干扰）
- **相位漂移**（温度变化导致信号链延迟变化）
- **载波失真**（励磁放大器非线性）

在这些情况下，峰值采样法容易因为采样点恰好在噪声尖峰上而产生较大误差，且如果相位延迟随温度漂移，峰值点的位置也会漂移，导致包络幅值波动。

而同步整流积分法通过在一个半周期内对信号进行平均，可以：

- **抑制高频噪声**：噪声的积分值趋于零。
- **容忍相位漂移**：只要载波极性判断正确，积分结果对采样点的具体位置不敏感。
- **利用更多信息**：充分利用了整个半周期的信号能量，而不是仅用两个采样点，信噪比更高。

### 4. 实现要点与挑战

- **采样率要求**：如果载波频率是10kHz，每个周期采16点，则ADC采样率需要达到160kHz以上，这对MCU的ADC模块和中断处理有一定压力，但大部分现代MCU（如STM32G4、TMS320F28系列）都可以轻松支持。
- **极性同步**：需要准确知道每个采样点属于载波的正半周还是负半周。通常可以用：
- 软件生成载波并输出PWM，同时用同一个定时器触发ADC采样，软件内部维护一个极性计数器。
- 或者硬件比较器检测励磁信号的过零点，但会增加外围电路。
- **直流偏置的影响**：积分法对直流偏置也比较敏感，因为直流偏置在半周期积分中不会抵消（正负半周期积分都会叠加），所以需要在硬件上做AC耦合或软件中扣除直流分量。
- **积分窗口**：可以选择半周期积分或全周期积分。全周期积分可以进一步抵消偶次谐波，但响应速度会变慢（一个周期更新一次包络 vs 半周期更新一次）。

### 5. 总结

**同步整流积分法**是旋变软解码中一种稳健的解调方法，它通过在载波半周期内对信号进行积分来提取包络，相比峰值采样法：

- **优点**：抗噪声能力强，对采样时序相位不敏感，信噪比高，适合恶劣电磁环境。
- **缺点**：需要更高的ADC采样率，计算量稍大（但仍是低开销的整数累加，无需乘法器或滤波器）。

在实际应用中，如果你的MCU有足够的ADC采样能力和处理能力，且现场干扰较大，同步整流积分法是比峰值采样法更可靠的选择。如果MCU资源紧张且硬件调理做得很好，峰值采样法更简单高效。两者都可以与后续的跟踪观测器（如反正切+PLL）配合，完成完整的旋变软解码。
