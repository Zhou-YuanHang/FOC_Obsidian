**电枢电流是指流经电机电枢绕组的电流，所以电枢电流不可以简单的理解为有功电流。**

# 一、概述

## 1. 核心定义：电枢是什么？

在解释电流之前，首先要明确**电枢**是什么。电枢是电机中**产生感应电动势**或**实现能量转换**的关键部件。

- **在电动机中**：电枢电流流入电枢，在磁场中受到电磁力的作用，从而产生转矩，驱动电机旋转（将电能转化为机械能）。
- **在发电机中**：原动机驱动电枢旋转，电枢绕组切割磁感线，从而产生电流输出（将机械能转化为电能）。

## 2. 关键区分：电枢电流 vs. 励磁电流

- **励磁电流**：是专门用来建立电机主磁场的电流（流经励磁绕组）。
- **电枢电流**：是参与能量转换的主力电流，也就是产生电磁力的电流。

## 3. 在不同电机中的具体表现

电枢电流的具体路径和性质取决于电机的类型：

- **直流电机**：

- 电枢通常指**转子**。
- 电流通过电刷和换向器流入转子绕组。
- 这个电流是交变的（在绕组内部），但从外部看是直流的。

- **交流同步电机**：

- 电枢通常指**定子**。x
- 电枢电流就是流入定子绕组的三相交流电。

- **交流异步电机（感应电机）**：

- 电枢通常也指**定子**。
- 定子电流既是建立磁场的电流，也是电枢电流（因为在感应电机中，转子电流是由电磁感应产生的）。

## 4. 电枢电流的作用与影响

- **产生转矩/功率**：电枢电流的大小直接决定了电机的输出转矩（对于电动机）或输出功率（对于发电机）。
- **产生磁场（电枢反应）**：电枢电流本身也会产生一个磁场，这个磁场会干扰和扭曲由励磁电流产生的主磁场，这种现象称为**电枢反应**。电枢反应会影响电机的运行性能。
- **发热**：电枢电流流过绕组时，由于绕组存在电阻，会产生热量（铜损）。因此，电枢电流的大小通常受到电机散热能力的限制（即额定电流）。

# 二、针对 PMSM

对于 PMSM 来说，电枢电流指的是流经三相绕组的电流，而且也不是全部用来对外做功的。

## 1、做功与不做功

PMSM 的转子磁场虽然是由转子永磁体产生的，不需要像异步电机那样从定子侧吸收无功电流来励磁，但是这也不代表着 PMSM 的电枢电流是全部用来做功的

# 2、解释

对于 PMSM 的控制，我们一般采取 FOC 控制，FOC 将电枢电流分解成了两个相互垂直的分量：

### 交轴电流

交轴电流产生垂直于转子磁场方向的磁场，用来产生电磁力矩

### 直轴电流

直轴电流产生与转子磁场方向相同的磁场。

- 情况1（Id = 0 控制）：在基速以下（额定转速以内），我们通常让![](https://cdn.nlark.com/yuque/__latex/d57ddee1e9f76ac96dba98967201b9c6.svg) 此时，电枢电流几乎全部是，全部用于出力。
- 情况2（弱磁控制）：当电机转速需要超过额定转速（基速）时，由于永磁体磁场恒定，随着转速升高，反电动势会越来越高（可能超过逆变器电压极限）。此时，我们需要通入负的![](https://cdn.nlark.com/yuque/__latex/0f11d0e3751321011665a76c82118736.svg)（即去磁电流），用来削弱永磁体的磁场，从而降低反电动势，让电机能继续升速
