# 1. 什么是功率因数？

**功率因数（PF）** = 有功功率 / 视在功率  
它代表了你使出的力气里，有多少是真正用来“干活”的。

- 功率因数越接近 **1**，说明效率越高，浪费的力气越少。
- 功率因数越低，说明浪费在“维持平衡”上的力气越多，电网的负担就越重。

# 2. 同步电动机的特殊之处

这里说的同步电机指的是电励磁同步电机，而不是永磁同步电机，所以需要励磁电流来产生转子磁场

一般的异步电动机，因为需要从电网吸收无功功率来建立磁场，所以功率因数总是小于1，而且是滞后的（电流滞后于电压）。

但同步电动机不一样：

- 它的转子磁场是由**励磁电流**产生的，而不是从电网“借”来的。
- 我们可以通过**调节励磁电流**的大小，来改变它从电网吸收或发出的无功功率，从而改变功率因数。

具体来说：

- **欠励磁**：励磁电流太小，电机从电网吸收滞后的无功功率，功率因数小于1。
- **正常励磁**：励磁电流刚好，电机不从电网吸收无功，功率因数等于 **1.0**，这是最理想的状态。
- **过励磁**：励磁电流太大，电机反而向电网发出超前的无功功率，功率因数大于1（超前）。
